  2. YAML 配置文件修改（loss_p_rn50.yaml）

  添加了详细的参数说明：
  - USE_QUERY_EMBED_TRAIN: true - 训练时使用可学习的 query embedding（默认）
  - USE_QUERY_EMBED_EVAL: false - 评估时使用冻结文本特征（默认，保证泛化能力）

  使用场景

  默认配置（推荐）：
  - 训练：USE_QUERY_EMBED_TRAIN: true - 学习更好的文本表示
  - 评估：USE_QUERY_EMBED_EVAL: false - 泛化到 novel 类别

  实验对比：
  你可以通过修改这两个参数来对比不同组合的效果，例如：
  - 都设为 false：完全使用冻结的 CLIP 文本特征
  - 都设为 true：训练和评估都使用可学习的 embedding（仅适用于 base classes 评估）
