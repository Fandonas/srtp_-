  阶段二：get_feats — CLIP视觉编码器提取帧特征

---
  阶段三：Temporal Transformer 增强帧特征

  位置：semantic_alignment_few_shot.py:505~529

  target_features [5, 5, 512] → context2(q=k=v自注意力) → [5, 5, 512]
  support_features [5, 5, 512] → context2(q=k=v自注意力) → [5, 5, 512]

  context2 是 Transformer_v1，在帧维度做自注意力，强化帧间时序关系，不改变形状。

---
  阶段四：合并 support + target 特征

  位置：semantic_alignment_few_shot.py:532

  cat([5,5,512], [5,5,512]) → all_features: [10, 5, 512]
                                （10个视频 × 5帧 × 512维）

---
  阶段五：文本特征提取（_create_stage_text_features）

  位置：semantic_alignment_few_shot.py:137

  以动作类别 "pick up then throw ball" 为例：

  "pick up then throw ball"
  ↓ re.split(r'[,，]\s*|then\s+', ...)
  → stages = ["pick up", "throw ball"]   （S = 2）

  每个阶段：
  "a video of pick up"
  → tokenize → [1, 77]
  → full_clip_model.encode_text() → [1, 512]

  cat([1,512], [1,512]) → stage_text_features["pick up then throw ball"]: [2, 512]

  所有类别的文本特征存储在字典 self._stage_text_features 中，每个类对应 [S_i, 512]。

---
  阶段六：计算语义对齐损失（_compute_semantic_alignment_loss_multi_class）

  位置：semantic_alignment_few_shot.py:252

  对 all_features 中每个视频 i（共10个），与 episode 内每个类 j（共5个）：

  frame_features = all_features[i]      → [5, 512]   （单视频5帧）
  stage_text_features[class_j]          → [S, 512]   （该类语义阶段）

  cos_sim([5,512], [S,512]) / temperature(0.1)
                                         → similarity_matrix: [5, S]
  1 - sigmoid(similarity_matrix)         → dists: [5, S]   （值∈(0,1)，越小越对齐）

  dists.unsqueeze(0).unsqueeze(0)        → [1, 1, 5, S]   （OTAM需要的4D格式）
  OTAM_cum_dist_v2([1,1,5,S], lbda=0.5) → [1, 1]  取[0,0] → 标量 cum_dist_j

  对 episode 5个类循环后：

  torch.stack([cum_dist_0,...,cum_dist_4]) → class_cum_dists: [5]

  logits = -class_cum_dists              → [5]   （距离小→logit大→概率高）
  gt_index = episode_classes.index(gt_name)  → 标量整数（如 2）
  F.cross_entropy(logits[None], gt_index[None])  → 标量损失

  对 batch 内10个视频取均值 → 最终 semantic_alignment_loss: 标量

       ↓ cos_sim / temperature
  相似度:        [T, S]               = [5, S]
       ↓ 1 - sigmoid
  距离矩阵:      [T, S]               = [5, S]
       ↓ unsqueeze × 2

       ↓ OTAM_cum_dist_v2

       ↓ stack × N_way=5
  类别距离:      [5]
       ↓ -logits → cross_entropy
  单视频损失:   标量
       ↓ .mean() over 10 videos
