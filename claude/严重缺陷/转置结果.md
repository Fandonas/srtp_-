  逐位置检查转移规则

  顶行 l=0（阶段0）：
  cum_dists[:,:,0,m] = dists[:,:,0,m] + cum_dists[:,:,0,m-1]  # 水平累加
  → 阶段0 横扫所有帧（帧0～T-1）。一个阶段对应多帧 ✓

---
  中间列 m=2..T（帧1～T-1），l≥1：
  cum_dists[:,:,l,m] = dist[l,m] - λ·log(
      exp(-cum[l-1,m-1]/λ) +   # 对角：前一阶段+前一帧 → 当前阶段+当前帧
      exp(-cum[l,m-1]/λ)        # 水平：同阶段+前一帧 → 同阶段+当前帧（一阶段多帧 ✓）
  )
  没有垂直项 exp(-cum[l-1,m]/λ) → 帧1～T-1 每帧只能对应一个阶段 ✓

---
  第1列 m=1（帧0），l≥1：
  cum_dists[:,:,l,1] = dist[l,1] - λ·log(
      exp(-cum[l-1,0]/λ) +   # 对角（从左padding）
      exp(-cum[l-1,1]/λ) +   # ← 垂直项！前一阶段+帧0 → 当前阶段+帧0
      exp(-cum[l,0]/λ)        # 水平（从左padding）
  )
  存在垂直项：帧0 可以被多个阶段匹配 ⚠️
